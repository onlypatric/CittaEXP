# 03 - Comandi Admin e Sistema

## `/admintown` (admin operations)
- `claim`, `unclaim` (anche su town target),
- `ignoreclaims`, `chatspy`,
- `delete`, `takeover`,
- `balance`, `setlevel`,
- `bonus`,
- `advancements`,
- `prune`.

Uso tipico:
- manutenzione server,
- recovery town,
- override amministrativi.

## `/husktowns` (plugin/system)
- `about`, `status`, `update`,
- `reload`,
- `dump confirm`,
- `migrate ...`.

## Policy consigliata per CittaEXP
- player non devono usare `/town` diretto se wrapper `/city` e l'interfaccia pubblica,
- staff tecnico puo avere bypass nativo per troubleshooting,
- runbook deve includere chiaramente quando usare `/admintown` invece di staff-overlay CittaEXP.
