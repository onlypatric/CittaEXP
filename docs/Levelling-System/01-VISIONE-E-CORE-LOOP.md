# 01 - Visione e Core Loop

## Visione
Il levelling città deve premiare:
- cooperazione tra membri,
- attività costante (non solo sessioni lunghe),
- varietà di gameplay (mining, farming, combattimento, economia),
- progressione stagionale leggibile.

## Core loop
1. La città riceve sfide attive (`daily`, `weekly`, `monthly/event`).
2. I membri contribuiscono con azioni di gioco reali.
3. Il progresso città avanza e genera XP/Level.
4. Il completamento sblocca bonus strutturali (tier, cap claim/membri, eventuali perk futuri).
5. La stagione si chiude e si riparte, mantenendo storico.

## Categorie principali
- Contributo risorse città (`/town contribute ...` o GUI equivalente).
- Mining globale città.
- Kill mobs globale.
- Farming globale.
- Sfide boss.
- Sfide commercio.

## Struttura sfide per ritmo
- Daily: obiettivi piccoli, XP bassa.
- Weekly: obiettivi principali, XP media.
- Monthly/Event: obiettivi grandi, XP alta.

## UX desiderata
- Stato città sempre visibile in GUI (progress bar + top contributor).
- Feedback frequente: progresso live, completamento sfida, reward ricevuta.
- Messaggistica breve e chiara per player giovani.
